# Emacs Cheet Sheet

## シェルとの行き来

    M-x shell
    C-x k

## ウィンドウ操作

    C-x 1 # たくさんに分かれたウィンドウを「ひとつに」まとめる
    C-x 2 # 「ふたつに」分割
    
## ジャンプ

    M-x goto-line # 指定した行にジャンプ
    C-s # 指定したキーワードにジャンプ。
        # もう一度C-sで次の候補
        # Enter で指定した場所にとどまる
    M-> # ファイルの最後
